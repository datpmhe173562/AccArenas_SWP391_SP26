# AccArenas Use Cases

| ID    | Use Case                     | Actor(s)                        | Use Case Description                                                                                                          |
| ----- | ---------------------------- | ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| UC-01 | Register Account             | Guest                           | Allows a guest to create a new account by providing personal information and verifying via email before accessing the system. |
| UC-02 | Login                        | Guest, Customer                 | Allows a registered user to authenticate using email and password to access system features.                                  |
| UC-03 | Forgot Password              | Guest, Customer                 | Allows a user to request a password reset link sent to their registered email.                                                |
| UC-04 | Browse Game Account Listings | Guest, Customer                 | Allows users to browse, search, and filter available game account listings and view account details without credentials.      |
| UC-05 | View Promotions              | Guest, Customer                 | Allows users to view active promotions, discounts, and vouchers.                                                              |
| UC-06 | Manage Profile               | Customer                        | Allows a customer to view and update personal profile information.                                                            |
| UC-07 | View Order History           | Customer                        | Allows a customer to view a list of past orders.                                                                              |
| UC-08 | View Order Detail            | Customer                        | Allows a customer to view detailed information of a selected order.                                                           |
| UC-09 | Purchase Game Accounts       | Customer                        | Allows a customer to select a game account and place an order.                                                                |
| UC-10 | Make Online Payment          | Customer, Payment Online System | Allows the system to send payment requests to the payment gateway and receive payment confirmation.                           |
| UC-11 | Receive Account Credentials  | Customer, Email Service         | Allows the system to securely deliver purchased account credentials and order confirmation via email.                         |
| UC-12 | Submit Feedback              | Customer                        | Allows a customer to submit feedback or reviews for purchased accounts.                                                       |
| UC-13 | Logout                       | Customer                        | Allows the customer to log out from the system securely.                                                                      |
| UC-14 | View Assigned Orders         | Sales Staff                     | Allows sales staff to view orders assigned to them by the system.                                                             |
| UC-15 | Monitor Order Fulfillment    | Sales Staff                     | Allows sales staff to monitor the delivery and fulfillment status of assigned orders.                                         |
| UC-16 | Update Order Status          | Sales Staff                     | Allows sales staff to update order status in exceptional cases (e.g., resend, refund).                                        |
| UC-17 | Handle Customer Inquiries    | Sales Staff                     | Allows sales staff to respond to customer inquiries and after-sales support requests.                                         |
| UC-18 | View Sales Statistics        | Sales Staff                     | Allows sales staff to view personal sales performance and statistics.                                                         |
| UC-19 | View Marketing Analytics     | Marketing Staff                 | Allows marketing staff to view analytics related to products, promotions, and customer engagement.                            |
| UC-20 | Manage Product Listings      | Marketing Staff                 | Allows marketing staff to create, update, and view game account product listings.                                             |
| UC-21 | Manage Promotions            | Marketing Staff                 | Allows marketing staff to create, update, delete, and view promotions and vouchers.                                           |
| UC-22 | Manage Blogs                 | Marketing Staff                 | Allows marketing staff to create, update, delete, and view blog posts and news content.                                       |
| UC-23 | Manage Banners               | Marketing Staff                 | Allows marketing staff to manage banners displayed on the system.                                                             |
| UC-24 | Manage Sliders               | Marketing Staff                 | Allows marketing staff to manage homepage sliders.                                                                            |
| UC-25 | Manage Account Categories    | Marketing Staff                 | Allows marketing staff to create, update, and view game account categories.                                                   |
| UC-26 | Manage User Accounts         | Admin                           | Allows admin to create, update, view, and manage system user accounts.                                                        |
| UC-27 | Re-assign User Role          | Admin                           | Allows admin to assign or change roles of system users.                                                                       |
| UC-28 | Manage Roles                 | Admin                           | Allows admin to create, update, delete, and view system roles.                                                                |
| UC-29 | View Financial Reports       | Admin                           | Allows admin to view system-wide financial and revenue reports.                                                               |
| UC-30 | Configure System Settings    | Admin                           | Allows admin to configure system parameters and operational settings.                                                         |
